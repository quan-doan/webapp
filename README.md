###demo介绍：
1. 横向滚动，置顶固定菜单；
2. 纵向滚动，支持滚动条；
3. 横屏竖屏宽度自适应；
4. 载入自动隐藏地址栏；
5. 锁定viewport；

###测试机器：
1. android2.3（本人自测）；
2. itouch4（本人自测）；
3. 帮忙测试的哥们可以在我的[博客](http://www.xiaoqiang.org/mobile-web/webapp-frame.html)留言反馈结果
